TwitterPhotoDownloader
======================

### English

Download **all** photo of a user from media page.

Using **Gecko** .

Requires **.NET Framework 4.5** or higher.

### Russian

Скачивает **все** фото пользователя с медиа страницы.

Используется **Gecko** .

Для работы требуется **.NET Framework 4.5** или выше.
